# 🎯 Valorant Crosshair Catalogue

Browse, preview, and copy popular Valorant crosshairs used by professional players and the community.

## 🔗 Live Site

**[valocrosshairbook](https://tinyurl.com/valocrosshairbook)** — Open instantly in your browser, no downloads needed.

## ✨ Features

- **Visual Previews** — Every crosshair is rendered accurately on an HTML5 Canvas
- **Spray Animation** — Hover over any crosshair to see its firing spread behavior
- **One-Click Copy** — Click "Copy Code" to grab the profile string for in-game import
- **Search & Filter** — Browse by Pro Players, Trending, or Fun/Custom categories
- **Background Toggle** — Switch between dark, light, and range backgrounds for visibility
- **4K/1440p Accurate** — Crosshairs are scaled to match real in-game proportions

Pure static HTML/CSS/JS — no frameworks, no build step, no dependencies.

## 📋 Usage

1. Visit the [live site](https://tinyurl.com/valocrosshairbook)
2. Browse or search for a crosshair you like
3. Click **Copy Code**
4. In Valorant: Settings → Crosshair → Import → Paste
